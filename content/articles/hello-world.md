# Hello, World

Welcome to this Astro-powered blog. This is a sample article.

## What is this?

This blog was migrated from a SvelteKit project to Astro 5, using:

- **Astro Content Collections** for articles and posts
- **Tailwind CSS v4** with OKLCH design tokens
- **React islands** for interactive components (theme, TOC)
- **i18n** in English and Traditional Chinese

## Features

- Dark mode with accent hue customization
- Rainbow mode animation
- Tags and categories
- Friends page
- Table of contents on article pages
